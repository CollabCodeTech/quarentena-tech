## Get Started

### Clone

- Clone this repo to your local machine using `https://github.com/CollabCodeTech/quarentena-tech.git`

```bash
cd quarentena-tech
cp .env.example .env
```

```bash
yarn
yarn dev
```
> or 
```bash
npm install
npm run dev
```

